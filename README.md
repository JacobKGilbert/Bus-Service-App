# Bus-Service-App
An app that brings diagnostics, management, and parental communication into a single platform.

### Reason For Making
Since I've started driving school bus for my local school district I've heard, and experienced, issues with our current applications. Seeing as I'll have the summer off, for the most part, from school and work, I wanted to find a project that could not only challenge me, but also keep me occupied.

### Current Applications
- Samsara
- Route Finder
- Stop Finder

### Main Features
- Route Planning
- Route Tracking
- Automatic Parental Notification
- Bus Diagnostics
- Bus Tracking
- Road Conditions
- In App Registration
- Route-by-Route Messaging

### Possible Stretch Goals
- Conditions Alter Route Timing
  - Parents are alerted to new estimated stop times.
